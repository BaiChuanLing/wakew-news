# 瓦客新闻网系统
瓦客新闻系统基于Java-web开发，纯JDBC+Servlet编写，无任何第三方依赖框架，一款简约而不简单的新闻系统。
# 使用工具
- 后台界面 -> beyond
- 提示框 -> x0popup
- 编辑器 -> WangEditor
- 下拉菜单 -> Select2
- IDE -> IntelliJ IDEA
- 头像上传 -> ajaxfileupload
- URL重写 -> urlrewrite
- 上传 -> Smartupload
- ...
